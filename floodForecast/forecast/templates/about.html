Final Year Project

The River Flood Forecasting Device is a smart IoT-based early warning system designed to monitor river water levels in real time and predict potential flood events using machine learning.

Sensor data is collected via an ESP32 microcontroller equipped with water level, temperature, humidity, rain, and wind sensors. The data is transmitted to this web dashboard where an XGBoost machine learning model analyses patterns and predicts future flood probability.

The system automatically sends Telegram alerts when water levels reach warning or danger thresholds, enabling faster emergency response and public safety measures.

System Specifications

  • Hardware: ESP32 Microcontroller
  • Sensors: Water Level, Temperature, Humidity, Rain, Wind
  • ML Model: XGBoost (Deployment 1)
  • Next Model: Temporal Fusion Transformer (TFT)
  • Backend: Django 6.0
  • Database: SQLite
  • Alerts: Telegram Bot
